## 🔧 Технические улучшения

### API Layer
- **fal-base-client.ts**: Добавлены progress callbacks и улучшенная обработка ошибок
- **fal-veo3-client.ts**: Автоматический fallback на бесплатные модели
- **Retry механизм**: Экспоненциальный backoff с умными сообщениями

### Bot Layer
- **dialog-fsm.ts**: Расширенная FSM с новыми состояниями
- **notification-service.ts**: Поддержка детального прогресса с временными метками
- **workflow-adapter.ts**: Интеграция с системой прогресса

### Workflow Layer
- **video-generation-workflow.ts**: Передача progress callbacks в реальном времени

## 📊 Новые возможности

### 🎛️ Режимы генерации
1. **Быстрый режим**:
   - Автоматическая генерация без промежуточных шагов
   - Без субтитров
   - Быстрый результат

2. **Детальный контроль**:
   - Просмотр и редактирование сценария
   - Проверка промптов перед генерацией
   - Опциональные субтитры
   - Полный контроль процесса

### 📊 Отслеживание прогресса
- **Этапы**: 8 различных этапов с эмодзи и описаниями
- **Временные метки**: Показ времени каждого обновления
- **Детали видео**: Позиция в очереди, процент выполнения
- **Статус сегментов**: Индивидуальный статус каждого видео

### 🔄 Обработка ошибок
- **Автоматический fallback**: Премиум → Бесплатные модели
- **Понятные сообщения**: Специфичные объяснения ошибок
- **Retry логика**: До 3 попыток с экспоненциальным backoff

### 📱 Команды бота
- `/start` - Начало работы с обновленным приветствием
- `/status` - Проверка текущего статуса генерации
- `/cancel` - Отмена активной генерации
- `/help` - Справка по командам
