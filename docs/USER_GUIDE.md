# 📱 Руководство пользователя Telegram бота

## 🚀 Быстрый старт

1. **Найдите бота** в Telegram: `@arendok_ai_grok_gen_bot`
2. **Нажмите** `/start` для начала работы
3. **Отправьте описание** видео, которое хотите создать
4. **Выберите параметры** по шагам
5. **Дождитесь результата** — бот покажет прогресс

## 📝 Как написать хорошее описание

### ✅ Хорошие примеры:
- "Кот играет с красным мячиком в зеленом саду"
- "Девушка пьет кофе в уютном кафе утром"
- "Закат над морем с летящими чайками"
- "Ребенок строит замок из песка на пляже"

### ❌ Избегайте:
- Слишком короткие описания: "кот"
- Сложные сцены: "битва драконов в космосе с лазерами"
- Неподходящий контент: насилие, политика, взрослый контент

## ⚙️ Выбор параметров

### 📏 Длительность
- **15 сек** — короткие клипы, быстрая генерация
- **30 сек** — оптимальный баланс качества и времени
- **60 сек** — длинные видео, больше деталей

### 📐 Соотношение сторон
- **9:16** — для Instagram Stories, TikTok, вертикальных видео
- **16:9** — для YouTube, горизонтальных видео

### 🎙️ Голоса
Выберите подходящий голос для озвучки:
- Мужские и женские варианты
- Разные тембры и стили

### 🤖 Модели
- **⚡ Премиум** — лучшее качество, может быть недоступно
- **🆓 Бесплатные** — всегда доступны, хорошее качество

## 🎛️ Режимы генерации

### 🚀 Быстрая генерация
**Когда использовать:**
- Нужен быстрый результат
- Доверяете боту создать сценарий
- Не нужны субтитры

**Процесс:**
1. Выберите параметры
2. Подтвердите генерацию
3. Ждите результат

### 🎛️ Детальный контроль
**Когда использовать:**
- Хотите контролировать сценарий
- Нужны субтитры
- Важно проверить промпты

**Процесс:**
1. Выберите параметры
2. Проверьте сценарий (можете отредактировать)
3. Проверьте промпты для кадров
4. Выберите субтитры
5. Запустите генерацию

## 📊 Отслеживание прогресса

Бот показывает прогресс в реальном времени:

```
🎬 Видео: 2/4 сегментов
Видео 2/4: IN_QUEUE (позиция 15)
⏰ 15:07:42
```

**Что означают статусы:**
- `IN_QUEUE` — ожидание в очереди
- `IN_PROGRESS 45%` — выполнение (45% готово)
- `COMPLETED` — сегмент готов

## 🔧 Полезные команды

- `/status` — проверить, что происходит с генерацией
- `/cancel` — отменить текущую генерацию
- `/help` — показать справку

## ⚠️ Что делать при ошибках

### "Проблема с доступом к API"
- Бот автоматически попробует бесплатные модели
- Попробуйте еще раз через минуту

### "Превышен лимит запросов"
- Подождите 2-3 минуты
- Попробуйте снова

### "Контент нарушает политику"
- Измените описание видео
- Избегайте спорного контента

### Генерация зависла
- Используйте `/status` для проверки
- При необходимости `/cancel` и попробуйте снова

## 💡 Советы для лучших результатов

1. **Будьте конкретными** в описании
2. **Используйте простые сцены** — они лучше генерируются
3. **Экспериментируйте с голосами** — найдите подходящий
4. **Попробуйте оба режима** — быстрый и детальный
5. **Используйте субтитры** для лучшей доступности
