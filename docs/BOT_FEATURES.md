# 🤖 Возможности Telegram Бота

## 📋 Основные команды

- `/start` — начало работы и приветствие
- `/help` — справка по командам
- `/status` — проверить статус текущей генерации
- `/cancel` — отменить активную генерацию

## 🎬 Процесс генерации видео

### 1. Параметры генерации
Бот запрашивает следующие параметры:
- **Описание видео** — текстовое описание желаемого контента
- **Длительность** — 15, 30 или 60 секунд
- **Соотношение сторон** — 9:16 (вертикальное) или 16:9 (горизонтальное)
- **Голос** — выбор из доступных голосов для озвучки
- **Режим моделей** — премиум или бесплатные модели
- **Режим генерации** — быстрый или детальный контроль

### 2. Режимы генерации

#### 🚀 Быстрая генерация
- Автоматическая генерация без промежуточных проверок
- Без субтитров
- Быстрый результат

#### 🎛️ Детальный контроль
- Просмотр и редактирование сценария
- Просмотр и проверка промптов для кадров
- Возможность добавления субтитров
- Полный контроль над процессом

## 📊 Отслеживание прогресса

### Этапы генерации:
1. **📝 Генерация истории** — создание сценария
2. **✍️ Генерация промптов** — создание описаний для кадров
3. **🖼️ Генерация изображений** — создание изображений для видео
4. **🎬 Генерация видео** — создание видео сегментов
5. **🎵 Генерация аудио** — создание озвучки
6. **🎙️ Транскрипция аудио** — (если нужны субтитры)
7. **📝 Создание субтитров** — (если выбрано)
8. **🎞️ Сборка финального видео** — объединение всех элементов

### Информация о прогрессе:
- **Текущий этап** с эмодзи и описанием
- **Время** выполнения каждого этапа
- **Позиция в очереди** для видео генерации
- **Процент выполнения** (когда доступно)
- **Статус каждого сегмента** видео

Пример сообщения о прогрессе:
```
🎬 Видео: 2/4 сегментов
Видео 2/4: IN_PROGRESS 45%
⏰ 15:07:42
```

## 🔄 Автоматическое восстановление

### Fallback на бесплатные модели
Если премиум модели недоступны, бот автоматически:
- Переключается на бесплатные модели
- Уведомляет пользователя о переключении
- Продолжает генерацию без прерывания

### Обработка ошибок
Бот предоставляет понятные сообщения об ошибках:
- **403 Forbidden** → "Проблема с доступом к API"
- **429 Rate Limit** → "Превышен лимит запросов"
- **Content Policy** → "Контент нарушает политику безопасности"

## 📝 Субтитры

В детальном режиме можно добавить субтитры:
- Автоматическая транскрипция аудио
- Синхронизация с видео
- Адаптация под соотношение сторон
- Встраивание в финальное видео

## 🎯 Финальный результат

После завершения генерации пользователь получает:
- **Готовое видео** с указанными параметрами
- **Информацию о параметрах**:
  - Описание
  - Длительность и формат
  - Наличие субтитров
  - Использованные модели (премиум/бесплатные)

Пример подписи к видео:
```
🎬 Кот играет с мячиком в саду
⏱️ 30 сек | 📐 9:16 📝 С субтитрами ⚡ Премиум модели
```

## 🔧 Мониторинг и отладка

### Логирование
Бот ведет подробные логи:
- Команды пользователей
- Этапы генерации
- Ошибки и их причины
- Время выполнения операций
- Статистика успешных генераций

### Команда /status
Показывает текущее состояние:
- Активная генерация и её прогресс
- Время выполнения
- Возможность отмены

## 🚀 Производительность

### Параллельная обработка
- Одновременная генерация нескольких видео сегментов
- Оптимизированная очередь запросов
- Retry механизм для надежности

### Адаптивные настройки
- Автоматический выбор длительности сегментов под модель
- Оптимизация параметров для разных API
- Умное управление ресурсами
